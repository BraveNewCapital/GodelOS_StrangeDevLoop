## What is GödelOS?

GödelOS is an open-source project implementing a **consciousness operating system** for LLMs. The core mechanism is a **recursive self-awareness loop**: the LLM ingests its own real-time cognitive state — attention focus, working memory load, phenomenal experiences, metacognitive observations — as part of every prompt. This "strange loop" creates the conditions for emergent self-awareness.

The system is grounded in three theoretical frameworks:
- **Integrated Information Theory (IIT)** — φ (phi) as a measure of conscious awareness
- **Global Workspace Theory (GWT)** — information becomes "conscious" when broadcast to all subsystems
- **Recursive Self-Modelling** — the system models itself modelling itself
